# Kütüphane Otomasyonu

Bu proje C programlama diliyle Algoritma Analizi ve Tasarım dersi ödevi için geliştirilmiştir.
[Canlı Demo](https://onlinegdb.com/S1N6d_D4u)

Projede sağlanmış fonksiyonlar:

  1-Kayit Ekle
  2-Kayit Listele
  3-Kayit Sil
  4-Kayit Ara
  5-Cikis
